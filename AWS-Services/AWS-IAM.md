## Identity and Access Management (IAM)
- Users and Groups
- Tags can be identified for each user
- Create your own password policy for users
- Provide users with access control to resources

## MFA Options in AWS
- Virtual MFA like Google Authenticator and Authy
- Universal 2nd Factor (U2F) Security Key 
- Harware Key Fob MFA Device
- Harware Key Fob MFA Device for AWS GovCloud (US)

## IAM Credentials Report (account-level)
- A report that lists all your account's users and the status of their various credentials

## IAM Access Advisor (user-level)
- Access advisor shows the service permissions granted to a user and when those services were last accessed.
- You can use this information to revise your policies.
